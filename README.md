# 🎯 Make 24 ゲーム

4つの数字を使って24を作る計算パズルゲーム！

## 🎮 遊び方

1. `index.html` をブラウザで開く
2. 表示された4つの数字を計算機ボタンで入力
3. 答えが24になればクリア！

## ✨ 機能

- **3つのレベル**: 
  - ふつう（18問）: 括弧なしで解ける問題
  - むずかしい（6問）: ×と括弧を使う問題
  - 鬼（6問）: 括弧と÷を使う問題
- **計算機ボタン**: ボタンで数字と演算子を入力（直接入力は無効）
- **問題ナビゲーション**: 前の問題・次の問題ボタンで自由に移動
- **解答例表示**: 困ったときは解答例ボタンで確認（正解率に影響）
- **採点機能**: 現在のレベルの正解数と正解率を確認
- **統計情報**: 正解率と連続正解数を表示
- **レベル別進行状況**: 各レベルの問題番号を個別に保持
- **美しいUI**: モダンでアニメーション付きのデザイン
- **レスポンシブ対応**: PC・スマホどちらでも快適にプレイ可能

## 📐 ルール

- 4つの数字すべてを1回ずつ使う
- 四則演算（+、-、×、÷）と括弧が使える
- 答えが24になればクリア！
- 解答例を表示した問題は回答不可（間違い扱い）

## 💡 例題

**レベル1（ふつう）**  
数字: 6, 6, 6, 6  
解答例: `6 + 6 + 6 + 6`

**レベル2（むずかしい）**  
数字: 1, 2, 3, 4  
解答例: `(1 + 2 + 3) * 4`

**レベル3（鬼）**  
数字: 8, 8, 3, 3  
解答例: `8 / (3 - 8/3)`

## 🎯 操作方法

- **数字ボタン**: 問題の4つの数字を入力（各1回のみ）
- **演算子ボタン**: +、-、×、÷、(、) を入力
- **BackSpace**: 最後の文字を削除
- **クリア**: 入力をすべて削除
- **= ボタン**: 答えを確認
- **⬅️ 前の問題**: 前の問題に戻る
- **➡️ 次の問題**: 次の問題に進む
- **📝 解答例**: 解答例を表示（正解率に影響）
- **📊 採点**: 現在のレベルの採点結果を表示
