# 🎯 Make 24 ゲーム

4つの数字を使って24を作る計算パズルゲーム！

## 🎮 遊び方

1. `index.html` をブラウザで開く
2. 表示された4つの数字を使って計算式を入力
3. 答えが24になればクリア！

## ✨ 機能

- **スコアシステム**: 正解するたびにポイント獲得
- **連続正解**: ストリーク（連続正解）でボーナスポイント
- **レベルアップ**: 3問連続正解でレベルアップ
- **ヒント機能**: 困ったときはヒントボタンで助けてもらえる
- **スキップ機能**: どうしても解けない問題はスキップ可能
- **美しいUI**: モダンでアニメーション付きのデザイン

## 📐 ルール

- 4つの数字すべてを1回ずつ使う
- 四則演算（+、-、*、/）と括弧が使える
- 答えが24になればクリア！

## 💡 例題

数字: 8, 8, 3, 3  
解答例: `8 / (3 - 8/3)`

数字: 1, 2, 3, 4  
解答例: `(1 + 2 + 3) * 4`
